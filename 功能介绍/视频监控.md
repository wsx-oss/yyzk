# 视频监控

## 1. 功能概述

视频监控模块是"智能工具人机交互与远程操控系统"的核心功能之一，用于管理和查看各监控区域的视频源信息。该模块支持视频源的新增、搜索、删除、播放等操作，并通过统计图表直观展示视频状态分布和区域覆盖情况，帮助管理人员全面掌握监控系统的运行状况。

## 2. 功能详细说明

### 2.1 视频监控信息管理

#### 新增视频源

用户可点击页面右上角的 **"新增视频源"** 按钮，打开新增视频源的弹窗。在该窗口中，用户需填写以下信息：

| 字段 | 是否必填 | 说明 |
| ---- | -------- | ---- |
| 视频名称 | 是 | 视频源的名称标识，如"A区入口监控" |
| 视频地址 | 是 | 视频源的播放地址（URL），如 `http://example.com/video1.mp4` |
| 监控区域 | 是 | 视频所属的监控区域：A区、B区、C区、D区 |
| 清晰度 | 否 | 视频清晰度等级：高清、标清、流畅 |
| 开始时间 | 是 | 视频监控的开始时间 |
| 结束时间 | 是 | 视频监控的结束时间 |
| 视频状态 | 是 | 当前视频源的运行状态：正常、故障、维护 |
| 是否录制 | 是 | 是否开启录制功能（开关控制） |

填写完成后，点击 **"确认"** 即可将新的视频源添加到系统中。数据会持久化保存到后端数据库，重启服务后不会丢失。

#### 刷新数据

点击 **"刷新"** 按钮，系统会重新从后端加载视频监控数据，确保页面展示的信息与数据库保持一致。

#### 删除视频源

- **单条删除**：在表格中，每条视频记录的操作列都有 **"删除"** 按钮，点击后弹出确认窗口，确认后即可删除该条记录。
- **批量删除**：用户可通过表格左侧的复选框选中多条记录，然后点击页面右上角的 **"删除选中"** 按钮，系统会弹出确认提示，确认后批量删除所有选中的视频源。

> **注意**：删除操作不可恢复，请谨慎操作。

#### 搜索视频

用户可在搜索表单中填写以下条件，点击 **"搜索"** 按钮进行筛选：

| 条件 | 说明 |
| ---- | ---- |
| 视频名称 | 按名称模糊匹配 |
| 视频地址 | 按地址模糊匹配 |
| 监控区域 | 按区域精确筛选（A区/B区/C区/D区/全部） |
| 视频状态 | 按状态精确筛选（正常/故障/维护/全部） |
| 清晰度 | 按清晰度精确筛选（高清/标清/流畅/全部） |
| 开始时间 | 筛选开始时间不早于指定时间的记录 |
| 结束时间 | 筛选结束时间不晚于指定时间的记录 |

#### 播放视频

在表格中，每条视频记录都有 **"播放"** 按钮，点击后页面会自动滚动到底部的视频播放窗口，并开始播放对应的视频。

### 2.2 视频监控统计分析

页面中部展示两个统计图表：

- **视频状态分布**（饼图）：以环形图形式展示正常、故障、维护三种状态的视频源占比，帮助用户直观了解监控系统的整体健康状况。
- **监控区域分布**（柱状图）：以水平柱状图形式展示各监控区域（A区、B区、C区、D区）的视频源数量，帮助用户了解不同区域的监控覆盖情况。

图表数据会随搜索筛选条件的变化而实时更新。

### 2.3 视频播放控制

页面底部提供专门的视频播放窗口：

- **播放窗口**：黑色背景的视频播放区域，用户点击表格中的"播放"按钮后，视频会在此窗口中显示。
- **播放控制按钮**：
  - **▶ 播放**：开始/继续播放视频
  - **⏸ 暂停**：暂停当前播放
  - **⛶ 全屏**：将视频切换为全屏显示

### 2.4 使用提示

页面中的使用提示区域提供以下操作建议：

1. 请确保输入的视频地址正确，否则可能无法正常播放。
2. 可通过搜索功能快速定位所需视频。
3. 删除视频源操作不可恢复，请谨慎操作。

## 3. 技术实现

### 3.1 后端接口

| 接口 | 方法 | 说明 |
| ---- | ---- | ---- |
| `/api/video/list` | GET | 获取所有视频源列表 |
| `/api/video/add` | POST | 新增视频源（JSON 格式） |
| `/api/video/:id` | DELETE | 删除指定 ID 的视频源 |

### 3.2 数据存储

后端使用 SQLite 数据库的 `video_sources` 表存储视频源信息，包含以下字段：

| 字段 | 类型 | 说明 |
| ---- | ---- | ---- |
| id | INTEGER | 主键，自增 |
| name | TEXT | 视频名称 |
| url | TEXT | 视频地址 |
| region | TEXT | 监控区域 |
| clarity | TEXT | 清晰度 |
| status | TEXT | 视频状态（正常/故障/维护） |
| recording | INTEGER | 是否录制（0/1） |
| start_time | TEXT | 开始时间 |
| end_time | TEXT | 结束时间 |
| created_at | DATETIME | 创建时间 |

### 3.3 前端技术

- 使用 Chart.js 绘制视频状态分布饼图和区域分布柱状图
- 使用 HTML5 `<video>` 元素实现视频播放
- 使用模态弹窗实现新增视频源和删除确认交互
- 搜索筛选在前端内存中执行，响应速度快

## 4. 使用指南

1. **查看视频源**：进入视频监控页面，系统自动加载并展示所有视频源。
2. **搜索视频**：在搜索表单中填写条件，点击"搜索"按钮。
3. **新增视频源**：点击"新增视频源"按钮，填写必要信息后点击"确认"。
4. **播放视频**：点击表格中的"播放"按钮，视频将在底部播放窗口中播放。
5. **删除视频源**：点击"删除"按钮或选中后点击"删除选中"，在确认弹窗中确认。
6. **刷新数据**：点击"刷新"按钮重新加载最新数据。
