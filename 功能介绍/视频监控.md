# 视频监控

## 1. 功能概述

视频监控模块是"智能工具人机交互与远程操控系统"的核心功能之一，用于集中查看和管理所有无人机的视频流。该模块与"无人机管理"模块深度联动——视频源不再独立管理，而是**直接从无人机注册表中读取**每台无人机配置的 `video_url` 字段。用户在注册或编辑无人机时填写视频流地址，即可在视频监控页面中统一查看、搜索、播放所有无人机的视频画面，并通过统计图表直观展示视频配置状态和协议分布。

> **架构变更说明**：早期版本使用独立的 `video_sources` 表管理视频源，现已废弃。当前版本的视频数据完全来源于 `drones` 表的 `video_url` 字段，实现了"一处注册、多处联动"的统一管理。

## 2. 功能详细说明

### 2.1 视频源数据来源

视频监控页面的数据来自 **无人机管理模块**（`/api/drones`）。每台无人机在注册时可填写 `video_url` 字段（视频流地址），填写后该无人机即出现在视频监控列表中。

- **已配置视频**：`video_url` 非空的无人机，在列表中显示为"已配置"，可直接播放
- **未配置视频**：`video_url` 为空的无人机，在列表中显示为"未配置"，播放按钮不可用

#### 如何配置视频源

1. 进入"无人机管理"模块
2. 点击"注册无人机"或编辑已有无人机
3. 在表单中填写 **视频流地址**（`video_url`）字段，例如：
   - HTTP 流：`http://192.168.1.100:8080/stream`
   - RTSP 流：`rtsp://192.168.1.100:554/live`
   - 本地文件：`http://192.168.1.100/video.mp4`
4. 保存后，视频监控页面会自动显示该无人机的视频

### 2.2 视频列表展示

页面以卡片或表格形式展示所有无人机的视频信息，包括：

| 列名       | 说明                                                   |
| ---------- | ------------------------------------------------------ |
| 无人机名称 | 无人机的注册名称                                       |
| 型号/序列号| 无人机的型号或序列号                                   |
| 视频地址   | 配置的视频流 URL                                       |
| 连接协议   | 无人机的远程连接协议（SSH/VNC/RDP）                    |
| 视频状态   | 已配置（绿色）/ 未配置（灰色）                         |
| 无人机状态 | 在线 / 离线                                            |
| 操作       | 播放按钮                                               |

### 2.3 搜索筛选

用户可在筛选区域中设置以下条件进行搜索：

| 条件       | 说明                                     |
| ---------- | ---------------------------------------- |
| 无人机名称 | 按名称模糊匹配                           |
| 视频状态   | 筛选已配置/未配置/全部                   |

### 2.4 视频播放

点击列表中的 **"播放"** 按钮，页面底部的视频播放窗口会加载并播放对应的视频流。

- **播放窗口**：黑色背景的视频播放区域
- **播放控制**：
  - **▶ 播放**：开始/继续播放视频
  - **⏸ 暂停**：暂停当前播放
  - **⛶ 全屏**：将视频切换为全屏显示

> **注意**：视频能否正常播放取决于视频流地址的有效性和浏览器对视频格式的支持。HTTP MP4/WebM 流可直接播放，RTSP 流需要通过转码服务转为浏览器可播放的格式。

### 2.5 视频统计

页面展示统计图表：

- **视频配置状态分布**（环形图）：展示已配置视频和未配置视频的无人机占比
- **连接协议分布**（柱状图）：展示不同连接协议（SSH/VNC/RDP）的无人机数量

### 2.6 使用提示

页面中的使用提示区域提供以下操作建议：

1. 视频源通过"无人机管理"模块统一配置，请在注册或编辑无人机时填写视频流地址。
2. 请确保视频流地址正确且可访问，否则可能无法正常播放。
3. 支持 HTTP/HTTPS 视频流直接播放，RTSP 流需配合转码服务使用。

## 3. 技术实现

### 3.1 后端接口

视频监控模块不再有独立的后端接口，直接复用无人机管理接口：

| 接口             | 方法 | 说明                                                   |
| ---------------- | ---- | ------------------------------------------------------ |
| `/api/drones`    | GET  | 获取无人机列表（含 `video_url` 字段，支持 name、status 筛选） |

### 3.2 数据存储

视频流地址存储在 `drones` 表的 `video_url` 字段中（`TEXT` 类型，默认为空字符串）。不再使用独立的 `video_sources` 表。

| 字段      | 类型 | 说明                                     |
| --------- | ---- | ---------------------------------------- |
| video_url | TEXT | 视频流地址（为空表示未配置视频）         |

### 3.3 前端技术

- 页面加载时调用 `/api/drones` 获取无人机列表，根据 `video_url` 是否为空判断视频配置状态
- 使用 Chart.js 绘制视频配置状态分布环形图和协议分布柱状图
- 使用 HTML5 `<video>` 元素或 `<iframe>` 实现视频播放
- 搜索筛选在前端内存中执行，响应速度快

## 4. 使用指南

1. **配置视频源**：进入"无人机管理"模块，注册或编辑无人机时填写视频流地址（`video_url`）。
2. **查看视频列表**：进入视频监控页面，系统自动加载所有无人机的视频配置信息。
3. **搜索视频**：在筛选区域输入无人机名称或选择视频状态，快速定位。
4. **播放视频**：点击"播放"按钮，视频将在底部播放窗口中播放。
5. **查看统计**：页面图表展示视频配置状态和协议分布。
