# 硬件状态检测

## 1. 功能概述

硬件状态检测模块是"智能工具人机交互与远程操控系统"的重要功能之一，用于实时监控和管理硬件设备的运行状态。该模块支持硬件设备的添加、搜索筛选、数据刷新、导出、查看详情、删除等操作，并通过统计图表直观展示硬件状态分布和硬件类型分布，帮助管理人员全面掌握硬件设备的运行情况。

## 2. 功能详细说明

### 2.1 硬件管理功能

#### 添加硬件

用户可点击页面右上角的 **"添加硬件"** 按钮，打开添加硬件的弹窗。在该窗口中，用户需填写以下信息：

| 字段     | 是否必填 | 说明                                                         |
| -------- | -------- | ------------------------------------------------------------ |
| 硬件名称 | 是       | 硬件设备的标识名称，如"主服务器"、"核心交换机"               |
| 硬件类型 | 是       | 设备类型：服务器、路由器、交换机、存储设备                   |
| 硬件IP地址 | 是     | 硬件设备的 IP 地址，如 `192.168.1.100`，需符合 IP 格式      |
| 初始状态 | 是       | 设备当前状态：在线、离线、警告                               |
| 描述信息 | 否       | 对该硬件设备的补充说明，如"二楼机房核心交换机"               |

> **说明**：IP 地址会进行格式校验，必须符合 `x.x.x.x` 的标准 IPv4 格式。

填写完成后，点击 **"确认添加"** 即可添加新的硬件设备。数据会持久化保存到后端数据库，重启服务后不会丢失。

#### 搜索硬件

用户可在搜索表单中填写以下条件，点击 **"搜索"** 按钮进行筛选：

| 条件         | 说明                                       |
| ------------ | ------------------------------------------ |
| 硬件名称     | 按名称模糊匹配                             |
| 硬件类型     | 按类型精确筛选（服务器/路由器/交换机/存储设备/全部） |
| 硬件IP地址   | 按 IP 地址模糊匹配                         |
| 检测时间范围 | 按检测时间范围筛选（开始日期 ~ 结束日期）   |
| 状态筛选     | 按状态精确筛选（在线/离线/警告/全部）       |
| 温度范围     | 按温度范围筛选（最小值 ~ 最大值，单位℃）   |
| CPU使用率范围 | 按 CPU 使用率范围筛选（最小值 ~ 最大值，单位%） |

> **说明**：所有筛选条件均为可选，支持组合筛选。未填写的条件不会参与过滤。

#### 刷新数据

点击 **"刷新数据"** 按钮，系统会触发后端重新采集硬件状态数据，并刷新列表和统计图表，获取最新的设备运行信息。

#### 导出数据

点击 **"导出数据"** 按钮，系统会将当前硬件列表中的所有数据导出为 CSV 文件（UTF-8 编码，含 BOM 头），方便在 Excel 中打开查看。导出字段包括：硬件名称、硬件类型、IP地址、状态、温度、CPU使用率、内存使用率、网络带宽、检测时间、描述。

### 2.2 硬件状态监测功能

#### 硬件列表

页面以表格形式展示所有硬件设备的详细信息，包括以下列：

| 列名       | 说明                                       |
| ---------- | ------------------------------------------ |
| 硬件名称   | 设备标识名称                               |
| 硬件类型   | 设备类型（服务器/路由器/交换机/存储设备）   |
| 硬件IP地址 | 设备 IP 地址                               |
| 状态       | 设备状态，以彩色圆点标识（绿色在线/红色离线/黄色警告） |
| 温度       | 设备温度（℃）                              |
| CPU使用率  | CPU 使用率（%）                            |
| 内存使用率 | 内存使用率（%）                            |
| 网络带宽   | 网络带宽使用情况                           |
| 检测时间   | 最近一次检测的时间                         |
| 操作       | 查看详情、删除按钮                         |

#### 设备状态显示

在硬件列表中，每个设备的状态以彩色圆点直观标识：

- **绿色圆点 + "在线"**：设备运行正常
- **红色圆点 + "离线"**：设备已断开或不可达
- **黄色圆点 + "警告"**：设备存在异常，需要关注

### 2.3 硬件详情功能

#### 查看详情

点击操作列的 **"查看详情"** 按钮，系统会弹出详情窗口，展示该硬件设备的完整信息，包括：

| 字段       | 说明                       |
| ---------- | -------------------------- |
| 硬件名称   | 设备标识名称               |
| 硬件类型   | 设备类型                   |
| IP地址     | 设备 IP 地址               |
| 状态       | 当前状态（含彩色圆点标识） |
| 温度       | 设备温度（℃）              |
| CPU使用率  | CPU 使用率（%）            |
| 内存使用率 | 内存使用率（%）            |
| 网络带宽   | 网络带宽使用情况           |
| 描述       | 设备描述信息               |
| 检测时间   | 最近一次检测时间           |
| 创建时间   | 设备记录创建时间           |
| 更新时间   | 设备记录最后更新时间       |

#### 删除硬件

点击操作列的 **"删除"** 按钮，系统弹出确认窗口，确认后将从系统中移除该硬件设备记录。

> **注意**：删除操作不可恢复，请谨慎操作。

### 2.4 硬件统计功能

页面底部展示两个统计图表：

- **硬件状态分布**（饼图）：以环形图形式展示在线、离线、警告三种状态的设备占比情况，用户可直观了解当前硬件设备的整体健康状况。
- **硬件类型分布**（柱状图）：以水平柱状图形式展示不同类型硬件（服务器、路由器、交换机、存储设备）的数量对比，帮助用户了解硬件资产的构成情况。

图表数据会随硬件列表的变化实时更新。

### 2.5 使用提示功能

页面中的使用提示区域提供以下操作建议：

1. 硬件状态检测用于实时监控硬件设备的运行状态，请定期刷新数据。
2. 若硬件状态显示为警告，请及时检查硬件设备。
3. 删除硬件信息后不可恢复，请谨慎操作。

## 3. 技术实现

### 3.1 后端接口

| 接口                          | 方法   | 说明                                                                                                     |
| ----------------------------- | ------ | -------------------------------------------------------------------------------------------------------- |
| `/api/hardware/items`       | GET    | 获取硬件列表（支持 name、type、ip、status、date_from、date_to、temp_min、temp_max、cpu_min、cpu_max 筛选） |
| `/api/hardware/items`       | POST   | 添加新硬件设备（JSON 格式）                                                                              |
| `/api/hardware/items/:id`   | GET    | 获取指定 ID 的硬件详情                                                                                   |
| `/api/hardware/items/:id`   | PUT    | 更新指定 ID 的硬件信息                                                                                   |
| `/api/hardware/items/:id`   | DELETE | 删除指定 ID 的硬件设备                                                                                   |
| `/api/hardware/items/stats` | GET    | 获取硬件统计数据（状态分布 + 类型分布）                                                                  |
| `/api/hardware/items/refresh` | POST | 触发硬件数据刷新                                                                                         |

### 3.2 数据存储

后端使用 SQLite 数据库的 `hardware_items` 表存储硬件信息，包含以下字段：

| 字段              | 类型     | 说明                                   |
| ----------------- | -------- | -------------------------------------- |
| id                | INTEGER  | 主键，自增                             |
| name              | TEXT     | 硬件名称                               |
| type              | TEXT     | 硬件类型（服务器/路由器/交换机/存储设备） |
| ip                | TEXT     | IP 地址                                |
| status            | TEXT     | 设备状态（在线/离线/警告）             |
| temperature       | REAL     | 温度（℃）                              |
| cpu_usage         | REAL     | CPU 使用率（%）                        |
| mem_usage         | REAL     | 内存使用率（%）                        |
| network_bandwidth | TEXT     | 网络带宽                               |
| description       | TEXT     | 描述信息                               |
| detected_at       | DATETIME | 检测时间                               |
| created_at        | DATETIME | 创建时间                               |
| updated_at        | DATETIME | 更新时间                               |

数据库为 `hardware_items` 表建立了以下索引以提升查询性能：
- `idx_hardware_items_name` — 硬件名称索引
- `idx_hardware_items_type` — 硬件类型索引
- `idx_hardware_items_status` — 设备状态索引
- `idx_hardware_items_ip` — IP 地址索引
- `idx_hardware_items_detected` — 检测时间索引

### 3.3 前端技术

- 使用 Chart.js 绘制硬件状态分布饼图和硬件类型分布柱状图
- 使用模态弹窗实现添加硬件和查看详情交互
- 搜索筛选通过后端 API 参数传递执行，支持多条件组合筛选
- 导出功能生成 CSV 文件，支持 UTF-8 编码和 BOM 头，兼容 Excel 直接打开
- IP 地址输入进行前端格式校验，防止无效数据提交
- 删除操作前弹出确认窗口，防止误操作

## 4. 使用指南

1. **添加硬件**：点击"添加硬件"按钮，填写硬件名称、类型、IP地址、状态等信息后点击"确认添加"。
2. **搜索硬件**：在搜索表单中输入筛选条件（名称、类型、IP、状态、温度范围、CPU范围等），点击"搜索"按钮快速定位设备。
3. **刷新数据**：点击"刷新数据"按钮，触发后端重新采集硬件状态，获取最新数据。
4. **导出数据**：点击"导出数据"按钮，将硬件列表导出为 CSV 文件保存。
5. **查看详情**：点击设备的"查看详情"按钮，在弹窗中查看完整的设备信息。
6. **删除硬件**：点击"删除"按钮，在确认弹窗中确认后移除设备记录。
7. **查看统计**：页面底部的图表实时展示硬件状态和类型分布。
