# 无人机状态监控专业知识

> 知识分类：领域专业知识 | 适用场景：状态监控、视频控制、异常告警

## 状态监控总体框架

无人机状态监控构建的是"飞行状态+设备状态+链路状态+环境状态+任务状态"的综合健康图谱。目标是尽早发现偏离、解释偏离来源、评估影响程度、给出处置建议。

状态监控五大类别：

1. 飞行姿态与运动状态（位置、速度、高度、航向、姿态角）
2. 能源与动力系统状态（电池电量、电压、温度、电机状态）
3. 通信与计算资源状态（遥控链路、图传链路、系统负载）
4. 传感器与载荷状态（GNSS、IMU、相机、云台）
5. 风险事件与异常告警状态（告警类型、严重级别、处置状态）

## 视频监控知识

视频监控既是任务载荷数据，也是态势感知的重要来源。

### 关键概念

- 视频流来源与编码方式（H.264/H.265/MJPEG）
- 主码流（高清录制存储）与子码流（低延迟实时预览）
- 时延影响：>200ms影响实时操控，>500ms严重影响决策
- 丢帧与码率抖动：影响视频AI分析准确度
- 视频流与位姿、航线的时间对齐（时间戳同步）

### 视频质量问题场景

低照度、逆光、雨雾、抖动、镜头起雾

### 视频问答

- 为什么视频画面延迟突然增大：可能是图传带宽下降、干扰增加、编码负载过高
- 为什么目标检测结果不稳定：可能是光照变化、抖动、低分辨率、目标遮挡
- 为什么视频正常但位姿更新异常：视频和飞控是独立通道，飞控数据可能延迟或丢失
- 视频中断时系统降级方案：切换子码流、降低分辨率、仅依赖遥测数据操控

## 远程桌面控制知识

远程桌面控制用于对地面站、边缘节点、仿真节点、运维主机的远程接入。

### 五个关键方面

1. **安全**：传输加密（TLS）、会话隔离、操作水印
2. **权限**：区分只读查看和完全控制；飞行控制操作需现场授权
3. **时延**：远程操作时延>100ms会影响操控精度；>300ms不建议做实时飞行控制
4. **审计**：所有远程会话录屏、操作日志、登录登出时间记录
5. **应急接管**：远程桌面失效时切换VPN直连或现场人员接管

### 权限边界

- 允许远程执行：查看监控、审核任务、调整参数、查看日志
- 必须现场授权：起飞/降落控制、紧急操控、固件升级、硬件维护

## 异常报警知识

异常报警是RAG最适合介入的模块之一。用户最常见的诉求是"这个告警是什么意思""严重吗""怎么处理"。

### 告警结构化知识对象

每条告警应包含：告警编号、告警名称、触发条件、数据来源、严重级别、影响范围、可能原因、推荐处置、是否需人工确认、是否可自动恢复、关联告警、历史处置案例。

### 告警分类

- **飞行安全类**：高度异常、姿态异常、速度异常、偏航异常
- **能源类**：低电、压差异常、温度异常、单体不均衡
- **导航类**：GNSS弱、RTK失锁、位置跳变、罗盘异常
- **链路类**：遥控失联、图传弱、时延过高、频段干扰
- **载荷类**：云台卡顿、相机掉线、传感器失真
- **平台类**：CPU过载、内存占用过高、磁盘不足
- **调度类**：任务冲突、资源冲突、权限异常
- **数据类**：时戳错乱、日志缺失、配置不一致

### 低电返航风险告警（典型条目）

告警名称：低电返航风险
定义：剩余电量低于任务继续执行与安全返航所需电量之和时触发。
主要触发指标：剩余电量百分比、瞬时电流、预测能耗、返航距离、风速修正系数。
常见原因：低温环境导致电压下坠、电池老化、逆风飞行、挂载过重、返航路径爬升过大。
风险：任务中断、迫降、失联。
推荐处置：立即评估最近备降点与返航点，必要时中止任务。
补充说明：不能仅根据剩余电量百分比判断，还需结合电压和负载变化趋势。

## 电池监控知识

### 电池基本参数

额定容量、标称电压、单体数量、放电倍率

### 实时监控指标

剩余电量(SOC)、总电压、单体压差、放电电流、温度、循环次数、健康度(SOH)

### 风险机理

- 老化：内阻增大，容量衰减
- 鼓包：内部化学反应异常，严重安全隐患
- 低温衰减：-10°C以下容量可能下降30-50%
- 高倍率放电：大机动动作导致电压塌陷
- 单体不均衡：某节电池提前耗尽导致整体保护

### 电池管理策略

- 起飞电量阈值：建议>90%
- 返航阈值：根据距离和风速动态计算，一般保留20-30%裕量
- 迫降阈值：<10%或电压持续下降
- 充放电窗口：20%-80%为理想存储范围
- 储存电量要求：长期存储保持50-60%

### 电池趋势型问答

- 同样35%电量，为什么今天更危险：可能因为低温、逆风、电池老化导致实际可用容量更低
- 低温环境电量显示正常但风险更高：低温导致电压曲线非线性下降，剩余百分比不反映真实可用能量
- 返航比去程更耗电：逆风、爬升返航路径、低电状态下电池效率降低
- 电池健康度下降的联动调整：降低任务距离限制、提高返航电量阈值、缩短单次任务时长

## 硬件状态检测知识

### 关键硬件组件

- **飞控**：不仅看在线状态，还要看输出一致性、控制延迟
- **IMU**：检查漂移、标定是否过期、震动是否超标
- **罗盘**：检查干扰、偏差补偿、多磁力计一致性
- **电机**：检查温升、转速一致性、电流负载异常
- **电调**：检查响应延迟、温度、校准状态
- **桨叶**：检查平衡、磨损、裂纹
- **云台**：不仅看画面是否有，还要看俯仰响应速度、回中误差、抖动程度
- **摄像头**：检查画面清晰度、对焦、白平衡
- **存储**：检查读写速度、剩余空间、坏块
- **天线**：检查信号强度、驻波比
- **散热系统**：检查风扇转速、散热片清洁度
- **边缘计算单元**：检查推理延迟、资源占用、热节流
