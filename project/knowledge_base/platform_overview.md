# CloudControl 智能无人机管控平台概述

> 知识分类：平台操作知识 | 适用场景：平台概览、功能导航、架构了解

## 平台简介

CloudControl 是一个综合性的智能无人机管控平台，采用 B/S 架构，后端使用 Go + Gin 框架，前端使用原生 HTML/CSS/JavaScript，支持 SQLite 和 MySQL 双数据库驱动。平台旨在提供无人机全生命周期管理，从注册、飞行规划到实时监控和异常处理。

## 系统架构

- **后端**: Go 语言 + Gin Web 框架，提供 RESTful API
- **前端**: 原生 HTML5 + JavaScript + Leaflet 地图 + Chart.js 图表
- **数据库**: 支持 SQLite（轻量部署）和 MySQL（云端部署）
- **AI 能力**: 集成大语言模型（LLM）用于智能路径规划和 AI 助手对话
- **实时通信**: WebSocket 用于实时数据推送和监控
- **任务调度**: 内置 goroutine 池，支持周期性巡检任务

## 核心功能模块

1. **无人机管理** - 无人机注册、编辑、删除，实时在线/离线状态管理
2. **飞行任务规划** - AI 智能航线规划、多方案对比、禁飞区绕行
3. **GPS 实时追踪** - 设备位置监控、历史轨迹回放、地理围栏告警
4. **电池监控** - 电量、温度、电压、健康度实时监测
5. **异常报警** - 多级告警系统（info/warning/critical），自动巡检
6. **视频监控** - 无人机实时视频流查看
7. **远程控制** - VNC/SSH/RDP 远程连接
8. **禁飞区管理** - 多边形和圆形禁飞区设置
9. **硬件状态检测** - 服务器和设备硬件监控
10. **仿真模拟引擎** - 批量无人机仿真、异常注入、性能测试
11. **强化学习训练** - 无人机集群协同 RL 训练
12. **CoT 思维链决策** - AI 推理过程可视化展示

## 部署方式

平台支持本地部署和云端部署。本地使用 SQLite 数据库，云端使用 MySQL（如 Aiven Cloud）。通过 `.env` 文件配置数据库连接、API 密钥等参数。
