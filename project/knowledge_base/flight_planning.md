# 飞行任务规划指南

> 知识分类：平台操作知识 | 适用场景：飞行规划、AI航线、任务创建

## AI 智能航线规划

平台集成大语言模型（LLM）进行智能飞行路径规划。规划流程：

1. 用户设定起点和终点坐标
2. 添加中间任务动作（拍照、悬停、巡查等）
3. 设置飞行约束（最大速度、最大高度、返航电量阈值）
4. 系统自动加载禁飞区数据
5. LLM 生成飞行计划，包含航点、动作、时间和电量估算
6. 几何引擎自动校正禁飞区违规
7. 验证器检查所有约束条件

## 多方案对比

系统支持同时生成三种飞行方案：

- **高效方案** (temperature=0.10): 最短路径优先
- **均衡方案** (temperature=0.35): 平衡距离和安全
- **保守方案** (temperature=0.65): 安全第一，创造性绕行

用户可以对比三种方案后选择最优路线。

## 禁飞区绕行

### 多边形禁飞区

- 航线不得穿越多边形边界
- 航点不得落入多边形内部
- 绕行时在多边形顶点外扩 150m 处插入绕行航点

### 圆形禁飞区

- 所有航点到圆心距离必须大于 (半径 + 150m)
- 航线不得穿越圆形区域
- 在圆外 150m 处规划切线绕行航点

### 几何自动校正

即使 LLM 规划的路线存在禁飞区违规，后端几何引擎会自动校正：

1. 第一轮：将落入禁飞区的航点推移到区域外部
2. 第二轮：使用可见图 + Dijkstra 算法寻找最短绕行路径
3. 后处理：贪心移除冗余航点，简化路线

## 飞行约束

- **最大速度**: 默认 8 m/s，可自定义
- **最大高度**: 默认 80m，受法规限制
- **返航电量**: 建议设置 30%，系统会在电量不足时告警
- **航点数量**: 控制在 5-20 个，避免冗余

## 降级方案

当 LLM 不可用时，系统自动使用直线降级规划：

- 起点直飞终点
- 中间动作按等距插入
- 仍然执行禁飞区几何校正
- 结果中会标注"降级规划"警告

## CoT 思维链决策

飞行规划支持 CoT（Chain of Thought）模式，LLM 会输出详细的推理步骤：

1. 分析任务需求和约束条件
2. 评估环境因素
3. 考虑无人机性能和电量限制
4. 制定多个候选方案
5. 比较各方案优缺点
6. 选择最优方案并说明理由

推理过程可在 CoT 决策页面查看（[NAV:cot]）。
