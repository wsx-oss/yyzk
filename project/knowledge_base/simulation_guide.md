# 仿真模拟引擎使用指南

> 知识分类：平台操作知识 | 适用场景：仿真引擎使用、批次管理、异常注入

## 概述

仿真模拟引擎允许用户批量创建虚拟无人机实例，进行各种测试和训练场景，无需真实硬件。

## 批次管理

### 创建批次

1. 前往仿真模拟页面（[NAV:simulation]）
2. 点击"创建批次"按钮
3. 设置参数：实例数量、飞行区域、飞行模式等
4. 启动批次后，虚拟无人机将自动模拟飞行行为

### 批次操作

- **启动**: 开始模拟飞行
- **暂停**: 暂停所有实例
- **停止**: 终止模拟
- **删除**: 清理批次数据

## 异常注入

仿真引擎支持向运行中的实例注入异常事件：

- **电量骤降**: 模拟电池突然放电
- **GPS 漂移**: 模拟 GPS 信号异常
- **通信中断**: 模拟与服务器断开连接
- **传感器故障**: 模拟各类传感器失效

注入异常后，可以观察系统的异常检测和告警响应能力。

## 强化学习训练

仿真引擎集成了强化学习（RL）训练模块：

- **训练目标**: 优化无人机集群的协同飞行策略
- **奖励函数**: 基于覆盖率、能耗、安全性等多维度指标
- **策略导出**: 训练完成后可导出策略并应用到新批次

查看训练指标：训练轮次、平均奖励、最佳奖励等。

## 禁飞区地理围栏感知

仿真引擎内置禁飞区检测能力：

- 启动时自动从 `no_fly_zones` 表加载多边形禁飞区数据
- 每个 tick 对所有运行实例执行 Ray-Casting 点在多边形内检测
- 违规实例的 `TelemetrySnapshot.GeofenceViolation` 置为 `true`
- RL 状态空间新增 `InNoFlyZone` 维度，违规时施加 -3.0 惩罚，纠正动作 +1.5 奖励

## 曲线轨迹显示

仿真地图支持平滑曲线轨迹显示：

- 实时轨迹：Catmull-Rom 样条平滑，避免折线感
- 航线展示：三次贝塞尔曲线，自适应弧度算法（短距小弧、长距大弧、急转弯加大曲率）
- Canvas 渲染器：大量无人机下保证地图性能
- 「曲线轨迹」开关：可在曲线/直线间切换对比

## 实时监控与可视化

仿真实例的状态可在地图上实时查看：

- 绿色标记：正常飞行中
- 黄色标记：有告警的实例
- 红色标记：故障/停止的实例
- 支持批次区域显示和聚合

## 仿真与真实系统的区别

仿真创建的虚拟设备会在系统的 GPS 设备和无人机表中创建记录。为区分仿真设备和真实设备：

- 仿真设备名称通常包含 "sim-" 前缀
- 仿真数据通过 sim_integration 模块写入
- 离线检测和告警系统对仿真和真实设备同等对待
