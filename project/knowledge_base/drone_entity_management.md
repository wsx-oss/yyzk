# 无人机实体建模与生命周期管理

> 知识分类：领域专业知识 | 适用场景：实体管理、设备建模、生命周期

## 无人机数字画像

每一架无人机应具备完整的设备数字画像，包含唯一标识和可追踪的生命周期信息：
- 基础信息：设备型号、序列号、机体类别、最大起飞重量
- 性能参数：续航能力、最大航程、巡航速度、最大爬升率、抗风等级
- 导航配置：GNSS配置（GPS/北斗/GLONASS/Galileo）、RTK支持
- 通信链路：遥控链路类型、图传链路类型、数传链路
- 载荷系统：云台规格、相机参数、载荷类型与重量
- 软件信息：固件版本、控制软件版本
- 运维记录：最近维护时间、累计飞行架次、累计飞行时长、故障记录、告警历史
- 权限归属：操作权限、所属部门、责任人

### 常见问答

- 该机型适合执行什么任务：需根据载荷能力、续航、抗风等级、速度综合判断
- 最大任务半径：取决于续航能力、通信链路距离、返航电量裕量
- 风险升高条件：高风速（超过抗风等级70%）、低温（<-10°C）、降水、强电磁干扰
- 常见硬件故障：电机温升过高、桨叶磨损、云台卡顿、IMU漂移、罗盘干扰
- 固件升级兼容性：需查看发布说明确认新增功能与已知问题
- 常见电池异常：单体压差过大、低温电压塌陷、循环次数超限、鼓包
- 必须搭配特定载荷的场景：测绘需RTK+倾斜摄影云台、安防需红外+可见光双光、物流需挂载系统

## 机型分类知识

### 按用途分类

| 类型 | 典型任务 | 关键能力要求 |
|------|----------|-------------|
| 巡检型 | 电力巡线、管道巡检 | 精确悬停、高清拍摄、自动航线 |
| 测绘型 | 地形测绘、三维建模 | 高精度定位(RTK)、大面积覆盖 |
| 安防型 | 周界巡逻、目标跟踪 | 红外载荷、快速响应、夜间作业 |
| 物流型 | 货物配送、应急物资投放 | 大载重、精确投放、航线稳定 |
| 应急型 | 搜救、灾害评估 | 快速部署、恶劣天气适应、长续航 |
| 侦察型 | 态势感知、情报收集 | 隐蔽性、高空作业、多传感器融合 |
| 集群协同型 | 大面积搜索、协同作业 | 通信组网、任务分配、防碰撞 |

### 按飞行平台分类

- **多旋翼**：悬停能力强、机动性高、起降要求低。适合精细巡检、近距离观测、城市安防、低空作业。主要不足：航时有限（通常20-45分钟），对电池状态高度敏感
- **固定翼**：航程长（可达数百公里）、覆盖面积大。适合大范围巡检与测绘。不足：起降条件要求高，低速精细悬停能力弱
- **复合翼（VTOL）**：兼具垂直起降和长航程优势。不足：系统复杂度高，对控制系统与维护水平要求更高

RAG检索应能根据用户提问自动关联"机型—能力—适用任务—限制条件—风险点"知识链。

## 生命周期管理

### 阶段与关键操作

1. **采购入库**：确认型号、配件清单、合格证书
2. **初始验收**：不应只关注"能否起飞"，还要核对：
   - IMU校准状态、指南针偏差
   - GNSS定位精度、RTK固定能力
   - 图传链路距离与延迟
   - 遥控链路可靠性
   - 云台稳定性与回中精度
   - 相机畸变校正
   - 存储卡写入速度
   - 返航逻辑测试
   - 低电策略验证
   - 避障功能测试
   - 自检日志完整性
3. **注册建档**：录入系统，建立数字画像
4. **启用配置**：设定飞行参数、返航高度、限速、围栏
5. **定期维护**：应记录以下详细信息：
   - 电池循环次数与健康度(SOH)
   - 桨叶磨损程度与更换记录
   - 电机温升测试结果
   - 机臂紧固检查
   - 天线状态与信号测试
   - 散热系统清洁状况
   - 接口腐蚀检查
   - 固件版本差异与升级记录
6. **部件更换**：更换云台需重新标定参数；更换电池需录入新电池信息
7. **故障维修**：记录故障现象、原因分析、维修方案、验证结果
8. **报废退役**：判定依据包括累计飞行时长、故障频率、关键部件寿命、维修成本

### 维护问答

- 累计飞行200小时后应重点检查：电机轴承、桨叶平衡、机臂碳纤维裂纹、云台减震球
- 电池循环300次后：建议降级使用（非关键任务）或退役
- 固件升级前：备份当前参数、航线模板、围栏配置、返航设置
- 更换云台后：重新标定俯仰零位、横滚补偿、航向跟踪参数
- 报废判定维度：结构完整性、飞控可靠性、通信稳定性、维修经济性、安全合规性
