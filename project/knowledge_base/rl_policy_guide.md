# 强化学习策略指南与决策知识

> 知识分类：领域专业知识 | 适用场景：RL策略、智能决策、训练优化

## RL 训练策略解读

### 算法概述

本平台采用 Tabular Q-Learning 结合 Experience Replay 进行无人机集群协同控制策略的训练。智能体在仿真环境中持续探索，学习在不同状态下采取最优动作的策略。

### 状态空间含义

RL 状态空间包含 20 维观测特征，分为三类：
- **单机状态**: 经纬度、高度、速度、航向、电量、温度、飞行阶段、任务进度、异常标志
- **集群上下文**: 附近无人机数量、最近无人机距离、集群质心位置
- **环境因素**: 风速、风向、时段

### 动作空间解释

| 动作编码 | 动作名称 | 适用场景 |
|---------|---------|---------|
| 0 | 调整航向 | 规避碰撞、修正航线偏离 |
| 1 | 调整速度 | 节约电量、追赶进度 |
| 2 | 调整高度 | 规避障碍、优化风阻 |
| 3 | 前往航点 | 正常任务推进 |
| 4 | 返航 | 低电量、任务完成、严重异常 |
| 5 | 悬停 | 等待指令、通信恢复 |
| 6 | 紧急降落 | 电池危急、严重故障 |

## RL 策略在真实决策中的应用

### 航线规划偏置

RL 训练结果可以作为先验知识注入到 LLM 航线规划中：
- **低电量场景**: RL 策略偏好返航/紧急降落 → 提示 LLM 规划时增加返航检查点
- **密集空域**: RL 策略偏好调整航向和高度 → 提示 LLM 增大航线间距
- **异常天气**: RL 策略偏好降速和降高度 → 提示 LLM 保守估算续航
- **碰撞风险**: RL 策略偏好规避动作 → 提示 LLM 在交叉区域错开高度

### 异常响应建议

基于 RL 多目标奖励函数学到的策略：

- **低电量异常 (battery_pct < 20%)**: 最优动作 = 返航或紧急降落；忌继续巡航或加速
- **航线偏离异常**: 最优动作 = 调整航向或前往航点修正；忌返航（除非偏离严重）
- **通信中断异常**: 最优动作 = 返航或悬停等待；忌继续执行任务
- **温度异常**: 最优动作 = 返航、紧急降落或调整高度（利用高空降温）
- **碰撞预警 (间距 < 30m)**: 最优动作 = 调整航向或高度规避；忌加速
- **无异常正常飞行**: 最优动作 = 前往航点推进任务；忌无故悬停或返航

### 集群协同策略

RL 学到的集群管理经验：
- 最优编队间距: 50-500 米（太近碰撞风险，太远协同效率低）
- 间距 < 10m 时应触发紧急规避
- 间距 10-30m 时应主动调整航向/高度
- 单机异常不应影响整个集群任务
- 优先级调度: 电量低的优先返航，任务进度高的优先完成

## 策略导出与应用

### Q-Table 解读

Q-Table 中每个状态键对应 7 个动作的 Q 值，Q 值越高表示该状态下执行该动作的长期收益越大。

### 策略应用到新批次

1. 导出当前训练好的策略 (rl_policy.json)
2. 创建新仿真批次时加载已有策略
3. 新批次中智能体直接利用策略行动（低 epsilon），同时继续微调

### 策略评估指标

- **路线效率**: 航点推进速度和任务完成率
- **安全分数**: 碰撞避免和围栏遵守
- **能源效率**: 电量管理和续航优化
- **异常处理**: 正确响应各类异常事件
- **集群协同**: 编队间距和整体任务完成率

## 从仿真到真实部署

### 仿真训练阶段

1. 在仿真环境中大量训练，探索各种场景
2. 注入异常事件测试策略鲁棒性
3. 评估收敛后的策略质量
4. 导出策略供真实部署参考

### 真实部署阶段

RL 策略不直接控制真实无人机，而是作为决策参考：
- 为 LLM 航线规划提供经验偏置
- 为异常处理提供推荐动作
- 为集群调度提供间距和优先级建议
- 为电量管理提供最优返航时机建议
