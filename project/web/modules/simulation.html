💡 强化学习训练说明

训练流程：启动训练后，RL智能体每2秒观察所有运行中无人机的状态（位置、速度、电量、异常等），选择动作（调整航向/速度/高度、跳航点、返航、悬停、紧急降落），实际执行到仿真环境中，然后观察下一状态并计算奖励。经验存入回放缓冲区，用Q-learning批量更新策略。

状态空间：GPS位置(经纬度100m网格)、高度(10m)、速度(5m/s)、航向(8扇区)、电量(10%)、飞行阶段、异常类型、附近无人机数量、最小间距。

奖励设计：路线效率20% + 安全避障20% + 能源管理15% + 任务完成15% + 异常处置20% + 编队内聚10%。奖励与动作强相关：低电量时返航+1.5分，忽略异常-1.0分等。

实机部署：训练收敛后，导出Q表策略文件(JSON)。实机控制器读取策略，根据实时传感器数据离散化状态后查表获取最优动作，通过MAVLink/DJI SDK执行。适用于巡检/巡逻等重复性任务的自主决策优化。
