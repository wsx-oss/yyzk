使用提示:
1. 数据源为远程设备IP，目标地址为本机。同步会从远程设备拉取数据到本地数据库。
2. 请确保远程设备已运行本系统（go run .），且两台设备在同一局域网内。
3. 全量同步会覆盖本地数据，增量同步只同步新增/更新的数据。
4. 同步内容包括：设备信息、无人机、电池监控、GPS定位、飞行任务、硬件状态、告警日志等。
