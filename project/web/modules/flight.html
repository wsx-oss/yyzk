使用提示
1. 创建任务后，可通过"状态跟踪"按钮控制飞行阶段：待命 → 起飞 → 巡航 → 执行任务 → 返航 → 降落。
2. 每次阶段变更会自动记录到任务日志中，可在详情中查看完整日志。
3. 支持批量导入/导出任务数据（JSON格式）。
