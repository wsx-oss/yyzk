无人机统一注册说明
1. 在此页面注册无人机后，系统会自动创建对应的远程控制设备和GPS追踪设备，无需在各功能模块重复添加。
2. 远程控制（SSH/VNC/RDP）：在“远程桌面控制”页面可直接选择已注册的无人机进行连接。
3. GPS/电池/飞行任务：只要无人机上运行了 hw-agent，数据会自动推送到平台，实时显示在各功能模块中。注册时无需填写经纬度，系统会在无人机 Agent 首次推送 GPS 数据时自动获取真实位置并回填。
4. Agent ID 是无人机上报数据的唯一标识。在伴飞计算机上运行 hw-agent -server http://主控端IP:8080 -id "你的AgentID" 即可关联。
5. 编辑或删除无人机时，关联的远程设备和GPS设备会自动同步更新或级联删除。
