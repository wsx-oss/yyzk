侧边栏导航

左侧菜单栏包含所有功能模块，按业务分组排列：

  • 飞行任务管理 — 禁飞区设置、航线规划、GPS 追踪、MAVLink 遥测
  • 仿真与智能决策 — 仿真引擎运行、CoT 思维链分析
  • 系统调度与运维 — 并发任务监控、数据备份与回滚
  • 无人机状态监控 — 视频监控、GPS 追踪、报警、电池、硬件检测
  • 日志管理 — 系统监控、性能分析、更新管理、操作日志
💡

核心操作指引

  • 注册无人机 → 进入「无人机管理」添加设备，填写名称与连接参数
  • 规划航线 → 进入「航线规划」，AI 自动生成航线并校验禁飞区合规
  • 启动仿真 → 进入「仿真模拟引擎」，创建仿真批次并启动 RL 训练
  • 智能分析 → 进入「CoT 智能决策」，对场景进行多步推理分析
  • 实时监控 → 查看 GPS 位置、电池状态、告警信息等实时数据
⚡

快捷提示

  • 点击右上角 月亮/太阳图标 可切换深色/浅色主题
  • 点击左上角 汉堡按钮 可折叠侧边栏，获得更大工作区
  • 右下角 AI 助手 可随时提问，获取智能建议
  • 右上角 铃铛图标 查看系统通知与告警消息

可随时通过右上角 ? 按钮重新打开本指南。
